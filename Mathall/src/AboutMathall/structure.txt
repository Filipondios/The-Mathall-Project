PROYECT STRUCTURE

The Mathall proyect is organized like this:

-> src ( Code )
	-> aboutMathall
		-> structure.txt (This file)
	-> algebraPackage
		-> AlgebraMatrices.java
		-> AlgebraEcuaciones.java
	-> filipondiosUtils
		-> FormattedNumber.java
		-> MatrixUtils.java
		-> Print.java
	-> graphicsPackage
		-> Graphics.java
	-> mathMainPackage
		-> Mathallmainpackage.java
	-> menusPackage
		-> MatricesMenu.java
		-> SistemasMenu.java
	-> userDataPackage
		-> UserData.java
		
-> dist (Mathall .jar executable)
	-> Mathall.jar
